==============================
Required Modules
==============================

 - Image Browser (http://drupal.org/project/imagebrowser)
 - jQuery UI (http://drupal.org/project/jquery_ui)
 - WYSIWYG API (http://drupal.org/project/wysiwyg)

==============================
Installation
==============================

 1. Enable the "Image Browser Plugin - WYSIWYG API" module (?q=/admin/build/modules)
 2. Edit a WYSIWYG profile (?q=admin/settings/wysiwyg) [NOTE: WYSIWYG API currently only supports Drupal plugins in the following editors: CKeditor, TinyMCE, FCKeditor]
 3. Enable the "Image Browser" button for your WYSIWYG profile and save changes.
